## How to run Application
<p>
  This is a command line application without input. You need to call functions in order to run it.

  1. Instantiate the new class, "event = Event()"
  2. Create a new event, "event.create_new_event(date, name)"
  3. Register attendee's for the event, "event.register_attendee(email, name)"
  4. When the event starts, check in attendee, "event.check_in_attendee(email, name)"
  5. Write a report, "event.generate_attendance_report"
</p>
